# Sorting-Algorithm-Visualizer


Visualization of sorting algorithm using javascript.  
Select new array of any size , then choose particular sorting button according to sort which you want to visualize.

## Algorithms Used
1.Bubble Sort  



2.Quick Sort  

3.Merge Sort  

4.Insertion Sort  

5.Selection Sort
